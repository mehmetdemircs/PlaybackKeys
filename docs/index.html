PlaybackKeys

Control web videos with keyboard shortcuts while you're working in another app. No switching windows. No breaking focus.

Install for Chrome

Quick Start

  1. Install the extension
  2. Open a video on YouTube, Vimeo, Udemy, or Coursera
  3. Press Ctrl+Shift+1 to pause/play — even if you're in another app
  4. Want different keys? Customize them at chrome://extensions/shortcuts

The Problem

You're coding along with a tutorial on your second monitor. You need to pause. So you alt-tab, click the video, pause, then switch back. 20 seconds later you need to rewind. Repeat.

Built-in browser shortcuts? Only work if the video tab is focused. Hardware media keys? They barely work on Udemy or Coursera.

PlaybackKeys lets you hit Ctrl+Shift+1 to pause from your editor. No switching. No clicking. Just one key and you're good.

Who Is It For?

  • Developers working through tutorials. Pause a 30-minute video without leaving your editor.
  • Students taking notes in Obsidian or Word. Control the lecture while you're writing.
  • People on Udemy/Coursera who've had enough of clicking back to the player.
  • Anyone who has a video on one monitor and work on the other, and is tired of context-switching.

Default Shortcuts

Windows / Linux macOS Action
Ctrl+Shift+1 ⌘+Shift+1 Play / Pause
Ctrl+Shift+2 ⌘+Shift+2 Speed +0.25x
Ctrl+Shift+3 ⌘+Shift+8 Skip back 5s
Ctrl+Shift+4 ⌘+Shift+9 Skip forward 5s

Need more? Assign speed down, reset speed, or switch target tab at chrome://extensions/shortcuts.

FAQ

Does it work on Chromebooks?

Nope. Global shortcuts aren't supported on ChromeOS, so they only work when Chrome is the active window.

Can I use it on other video sites?

Yep. Click the extension icon and hit "Enable on this site" for any site with an HTML5 video player. Chrome will ask for permission that one time.

Shortcuts aren't working. What's wrong?

Usually it's because Windows or macOS already uses that shortcut. Go to chrome://extensions/shortcuts and pick a different one. Make sure it's set to "Global," not "In Chrome."

What's it NOT going to do?

It won't skip ads, bypass DRM, change the player UI, download videos, or spy on you. It just pauses, skips, and changes speed. That's the whole thing.

What about Edge?

Not officially in the Edge store yet, but it should work since Edge uses Chromium. Let me know if you want it there.
